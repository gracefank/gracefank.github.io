Overview

Samartian is a web application that allows users to post and respond to favors based on location to connect neighbors and promote a sense of community.

My main responsibilities for this project were designing and implementing the user interface in HTML and CSS, and integrating the Google Maps API for location services.

Challenges

With this app, we wanted to use technology to reduce the disconnected urban lifestyle by bringing us closer our neighbors and allowing us to ask them for something as simple as lending sugar.

Although initially I had created this interface using Node.js, there were complications with integrating front and back-end together. As the team decided to switch to a web application, I reimplemented the interface in HTML and CSS. We wanted users of all age to be able to use our product, so I decided on a simple interface with as few buttons as possible.
